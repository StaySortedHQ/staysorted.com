Sorted’s URL Scheme

What is a URL Scheme?

URL Schemes allows for users to open an app and perform specific actions in the app. 

Sorted has its own URL Scheme to perform actions such as opening a specific view or adding in reminders. 

Instructions to use Sorted’s URL Scheme below: 
1. Add Reminder

staysorted://x-callback-url/add?title=…[&label=…][&date=(yyyy-MM-dd HH:mm)]

Properties:
title: (required) and should not be an empty

label: (optional) and should be the title of the label
   - if not specified or if empty, then reminder will go to inbox
   - if specified, will match existing labels (case-insensitive)
   - if no labels match, a new label will be created

date: (optional) can be the format yyyy-MM-dd or yyyy-MM-dd HH:mm

Example:
staysorted://x-callback-url/add?title=Buy%20Eggs&list=errands&date=2017-10-31%18:30


Past the URL Scheme in Safari Browser

Notification Shown to open in Sorted

“Buy Eggs” has bee successfull added to Sorted


2. Open Sorted to a specific view

staysorted://x-callback-url/open?item=...

Properties:
item: (required) and should be one of the following values
- today
- lists
- inbox
- search

Example:
staysorted://x-callback-url/open?item=today 


Past the URL Scheme in Safari Browser

Notification Shown to open in Sorted

Sorted Today View has now been shown
